Super Image tools is a image manage plugin

== Description ==

<p>Super Image tools is a image manage plugin, it contain following funticons:</p>

1. It can add watermark to image when you upload image in WordPress.
2. It can add two kinds of watermark, picture and text.
3. It can add watermark to all image in your WordPress.
4. It can download remoted image and add watermark.
5. add the downloaded image to a wordpress attachment.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Setting panel to configure plugin output.
